## 填充字符串

实现 mask 函数将"123456"转为"##3456",只保留最后四个字符

```js
const mask = (str, maskChar = '#') => {
  // 代码
};
console.log(mask('123456')); // '##3456'
console.log(mask('lindaidai')); // '#####idai'
```

### 代码

```js
const mask = (str, maskChar = '#') => {
  // 输入参数的边界情况
  if (!str || str.length <= 4) {
    return str;
  }

  // 代码
  const length = str.length;
  let prefix = '';
  for (let i = 0; i < length - 4; i++) {
    prefix += maskChar;
  }
  return prefix + str.slice(-4);
};

console.log(mask('123456')); // '##3456'
console.log(mask('lindaidai')); // '#####idai'
```
